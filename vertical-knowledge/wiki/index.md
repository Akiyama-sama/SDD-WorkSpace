# 模块清单

这里是领域知识的入口索引。模板阶段不预置具体业务模块，你可以按自己的项目逐步补充。

## 推荐分法

| 模块 | 适用场景 |
|------|----------|
| 核心流程 | 登录、下单、结算、发布等关键主链路 |
| 支撑系统 | 配置、权限、通知、任务、审计等辅助模块 |
| 数据视图 | 列表、报表、搜索、筛选、统计页 |
| 基础设施 | 路由、状态管理、接口封装、埋点、监控 |

## 维护建议

- 先写模块 `index.md`，再决定是否拆更细子页。
- 一条知识只描述一个主题，避免巨石文档。
- 每次变更结束后，只回写被这次需求真正影响的知识。
